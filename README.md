
## Run Locally

Clone the project:

```bash
  git clone https://github.com/vardhin/chatapp
```

Go to the project directory:

```bash
  cd chatapp
```

Install dependencies using pip:

```bash
  pip install twisted
  pip install pycryptodom
  pip install cryptography
```

To deploy and run this project:

*for Linux/Windows:*
```bash
  python node.py
```
*for MacOS:*
```bash
  python3 node.py
```
